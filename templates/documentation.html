Getting Started

Welcome to Rules Central! This documentation will help you get up and running quickly.

  1. Create a virtual environment and install the dependencies.
  2. Sign up and create your profile.
  3. Upload your first diagram or rule set.
  4. Explore the dashboard and metrics.

The refreshed Bear Light theme applies a brighter palette across all UI elements. Use the toggle or visit Settings to enable it.

Uploading Diagrams

Use the Upload page to add new diagrams or rule sets to your catalog.

  1. Open the Upload page from the sidebar.
  2. Drag & drop JSON or Mermaid files, or click to select files.
  3. Monitor the progress indicator for upload status.
  4. After completion, diagrams appear in the catalog automatically.

Exploring the Catalog

The catalog lets you browse, search, and filter all available diagrams.

  • Use the search bar to find diagrams by name or keyword.
  • Apply type and tag filters to narrow results.
  • Click a diagram card to open it in the viewer.
  • Bookmark favorites for quick access later.

Best Practices

  • Keep your rules modular and well-documented.
  • Use the search and filter features to quickly find what you need.
  • Collaborate with your team using the built-in tools.

User Roles & Permissions

Rules Central supports basic role-based access control.

  • Admins manage users and system settings.
  • Editors can upload and modify diagrams.
  • Viewers explore diagrams without editing rights.

Configuration

Customize the app with these environment variables:

  • CONFIG_PATH – path to an alternate config.json
  • SECRET_KEY – override the generated secret
  • UPLOAD_FOLDER – location for uploaded files
  • DIAGRAMS_FOLDER – directory for diagrams
  • PORT – port for wsgi.py

Troubleshooting

If you encounter issues, try these steps before reaching out.

  • Verify that uploads are valid JSON and under 10 MB.
  • Refresh your browser if diagrams fail to display.
  • Check the logs in the Settings menu for error details.
  • Set the CONFIG_PATH environment variable if you need a custom configuration file.
  • Contact support@rulescentral.com for persistent problems.
